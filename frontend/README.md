# Frontend

Aplicacao React da Sprint 0 do IWR PDV.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Variaveis de ambiente

- `VITE_API_BASE_URL`: URL base da API. Deixe vazio em desenvolvimento para usar o proxy do Vite.

## O que esta pronto

- pagina inicial da fundacao tecnica
- consumo do endpoint `/health`
- servicos HTTP centralizados
- estrutura separada entre pages, components e services
- tela de gestao de produtos
- busca, cadastro, edicao e ativacao/inativacao
- geracao automatica de codigo no padrao `IWR-000001`
- exibicao do QR Code do produto na listagem
- preview e impressao de etiqueta do produto
- tela de caixa com leitura por codigo e carrinho local
- finalizacao de venda com baixa de estoque
- historico de vendas com filtros basicos
